## 操作流程

### 搜索职位

1. 点击首页顶部搜索框
2. 输入职位名称或关键词（如"店长"、"经理"）
3. 点击搜索按钮或键盘上的"搜索"
4. 等待搜索结果加载（等待 2-3 秒）
5. 在结果列表中找到目标职位

### 发起沟通

1. 进入目标职位详情页
2. 点击"立即沟通"按钮
3. 等待跳转到聊天页面
4. 如有快捷打招呼语，可直接点击发送
5. 或手动输入打招呼内容后点击发送

### 查看消息

1. 点击底部导航栏"消息"标签
2. 在聊天列表中找到目标会话
3. 点击进入聊天详情页
4. 查看聊天记录

### 投递简历

1. 进入职位详情页
2. 点击"立即沟通"或"投递简历"
3. 确认简历信息
4. 提交投递

### 编辑简历

1. 点击"我的"进入个人中心
2. 点击"简历"入口
3. 编辑个人信息、工作经历、期望职位等
4. 保存修改

## 注意事项

- 首页和列表页使用 UITableView/UICollectionView 结构，可通过 class name 策略定位
- 首次打开可能会有登录弹窗或权限弹窗（定位、通知），需要先处理
- 聊天页面输入框可能需要先点击才能弹出键盘
- 部分操作需要登录状态，未登录时会跳转到登录页
- 搜索时可能需要等待网络请求（1-2 秒）
- 底部导航栏通常有 4-5 个标签，使用 name 策略定位
