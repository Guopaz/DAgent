
增强型 Agent 主循环
```mermaid
graph TD
    START((任务启动)) --> INIT[init: 初始化环境]
    INIT --> CLEANUP[cleanup: 前置清理弹窗/键盘]
    CLEANUP --> PLAN[plan: LLM 任务规划]
    PLAN --> REFRESH[refresh: 刷新屏幕状态]

    REFRESH --> PRE_CHECK{pre_check: 前置检测}
    PRE_CHECK -->|有弹窗| HANDLE_ALERT[自动处理弹窗]
    HANDLE_ALERT --> REFRESH
    PRE_CHECK -->|有键盘| DISMISS_KEYBOARD[自动关闭键盘]
    DISMISS_KEYBOARD --> REFRESH
    PRE_CHECK -->|加载中| WAIT_LOADING[等待加载完成]
    WAIT_LOADING --> REFRESH
    PRE_CHECK -->|正常| DECIDE

    DECIDE[decide: LLM 决策下一步] -->|有操作| EXECUTE
    DECIDE -->|无操作| VERIFY

    EXECUTE[execute: 执行工具调用] --> VERIFY

    VERIFY{verify: 三级验证}
    VERIFY -->|passed| NEXT_STEP{还有步骤?}
    NEXT_STEP -->|是| REFRESH
    NEXT_STEP -->|否| SUMMARY
    VERIFY -->|failed| RECOVERY

    RECOVERY[recovery: LLM 恢复分析]
    RECOVERY -->|retry| REFRESH
    RECOVERY -->|skip| NEXT_STEP
    RECOVERY -->|abort| SUMMARY
    RECOVERY -->|timeout| SUMMARY

    SUMMARY[summary: 生成报告] --> END((任务结束))
```
